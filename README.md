# Guess The Number Game

## English
This is a simple "Guess The Number" game written in C.  
Try to guess the secret number chosen by the computer.

### How to Compile and Run
1. Compile the program using:  
   `gcc main.c -o guess`
2. Run the executable:  
   `./guess`
